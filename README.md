# Bank-marketing-data-analysis
This repository contains an exploratory data analysis (EDA) and inspection workflow for the Bank Marketing Dataset, commonly used to predict whether clients will subscribe to a term deposit following a marketing campaign.

The notebook walks through data loading, cleaning, exploratory analysis, and basic preprocessing aimed at understanding customer behavior and key predictive features.

## 
Dataset loading and inspection
Missing value checks
Summary statistics
Insights into feature distributions
Early steps toward classification modeling

## Key Analysis Topics
Data Understanding
Preview of dataset structure
Examination of customer attributes (age, job, marital status, etc.)
Analysis of campaign-related features (contact type, number of contacts, previous outcomes)

Data Quality Assessment
Checking missing values
Inspecting data types
Identifying potential inconsistencies

Exploratory Analysis
Summary statistics for numeric variables
Distribution exploration
Feature relevance discussion
Initial insights on deposit subscription patterns
